# alarme
Um Projeto de alarme feito em Java, onde o usuário pode definir um horário para o alarme tocar. Um contador é iniciado no terminal até chegar no horário definido pelo usuário, e assim que chegar nesse determinado horário, o sistema iniciará o áudio do alarme até o usuário clicar na tecla "ENTER" do teclado. O áudio está pré-definido nos arquivos.
